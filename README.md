# JOGO-HIP-DOG-RUN-
🐶 Hip Dog Run é um jogo educativo inspirado no Chrome Dino Runner que promove a conscientização sobre a saúde articular dos cães. O cachorro corre automaticamente, e o jogador deve coletar itens benéficos à mobilidade e evitar obstáculos que possam prejudicar as articulações e a qualidade de vida do animal.
